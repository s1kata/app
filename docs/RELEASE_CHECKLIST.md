# Ручная проверка перед релизом (iOS / Android)

Используйте **production** или **preview** билд на реальном устройстве (не Expo Go), с production URL и секретами EAS.

## Поиск туров (LTE / release)

- [ ] Форма поиска: города вылета/страны/курорты загружаются до нажатия «Найти» (курорт не сбрасывается).
- [ ] После «Найти» — переход на результаты, loader на экране результатов (`runSearch: true`, единый путь `searchTours`).
- [ ] Жёсткие фильтры (курорт, 5★, ребёнок, 10–20 ночей): **результаты с первого раза**, без необходимости повторного поиска.
- [ ] Пустой ответ API (`[]`) — экран «Ничего не найдено», не ошибка.
- [ ] Невалидный meal не уходит в API; fallback питания id 2,3,4,5,7,9.
- [ ] Polling с `operatorStatus=false`; на cellular интервал ≥6 с, без параллельных poll-запросов.
- [ ] Список результатов: фото через `CachedImage`, плавный скролл.

## Навигация и UI

- [ ] Таб-бар и переходы на экран бронирования без артефактов.
- [ ] Модальные окна: «Назад» на Android; статус-бар после модалок страны.
- [ ] Блок «Персональные рекомендации» скрыт (`RELEASE_HIDE_NEXT_PATCH_UI`).

## Оплата

- [ ] Диалог перед внешней оплатой отображается.
- [ ] Deep link / закрытие браузера — корректный статус бронирования.
- [ ] Сеть: верхний баннер при проблемах (не блокирует UI); фоновые ретраи; toast «Соединение восстановлено»; данные на главной обновляются.

## Нативные возможности

- [ ] Локальные уведомления: разрешение в iOS → напоминание в 12:00 (`IOS_ENABLE_PUSH=1`).
- [ ] Геолокация: запрос разрешений и отказ.
- [ ] Длинные списки туров: прокрутка без сильных лагов.

## Конфигурация

- [ ] В билде нет `localhost` в URL оплаты и сайта.
- [ ] Production/preview: нет `TOURVISOR_TOKEN` и `EXPO_PUBLIC_UON_API_KEY` в бандле.
- [ ] `version` / `versionCode` / `buildNumber` увеличены перед submit.

## Store readiness

- [ ] Карточки магазинов: описание, скриншоты, https://travelhub63.ru/privacy.html, support email.
- [ ] Apple: Developer Program, сертификаты, privacy questionnaire.
- [ ] Описание оплаты соответствует факту (внешний веб-платёж, не IAP).
